# Simulador Juro Simples
## Descrição
Este projeto é um simulador para calcular juros simples.

## Como usar
1. Abra o link no seu navegador (PC ou Telemóvel)
2. Insira a Data de Inicio e Data de Fim (Periodo)
3. Insira a taxa de juro
4. Insira o capital
5. O resultado será calculado automaticamente

- `index.html` - Página principal do simulador
